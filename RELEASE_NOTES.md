# Release Notes

Version v0.1.6 — June 26, 2026

## Officers always get the guild + roster
Claiming a guild now always shares its GW2 key and guild (read-only) with everyone
in the workspace — that's the whole point of a workspace — so invited officers see
the live roster automatically, no setup. Sharing the AxiTools key is now the
optional toggle: leave it off and each officer adds their own key for Discord
features, or turn it on to hand yours to the whole workspace.

## Live invites and revokes
The member list and the sent/pending invite lists now update in real time. Invite
someone and it shows up; revoke it and it disappears — on both sides, no refresh.
